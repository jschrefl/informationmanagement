Example Internet Explorer:

The reason why web-developers don’t like IE is because they (=Microsoft) did not follow any standards for a long time. IE9 now follows some standards but does not support all features yet. If a browser does not follow certain rules how HTML and CSS code is interpreted a web-page will look different of every browser. Therefore web-developers have to write code again, just so that their website (that already looks great on Firefox, Chrome, Safari and Opera) looks great on Internet Explorer. Resume: Developers need more time (and money) to develop websites, only because Microsoft doesn’t follow all the web-standards.
